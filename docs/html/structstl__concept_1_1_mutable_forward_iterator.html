Detailed Description

template<typename It>
struct stl_concept::MutableForwardIterator< It >

A MutableForwardIterator is a ForwardIterator that additionally satisfies the OutputIterator requirements.

Requirements

The type It satisfies MutableForwardIterator if

  • The type It satisfies ForwardIterator
  • The type It satisfies OutputIterator
  • The type boost::iterator_reference<It>::type must be exactly T&, where T is the type denoted by boost::iterator_value<It>::type
