Returns the first element in the range [first, last) that predicate p returns true.

Template Parameters
InputIt- must meet the requirements of stl_concept::InputIterator.
UnaryPredicate- must meet the requirements of stl_concept::UnaryPredicate.
Parameters
first,last- the range of elements to examine
p- unary predicate which returns â€‹true for the required element.
The signature of the predicate function should be equivalent to the following:
bool pred(const Type &a);
The signature does not need to have const &, but the function must not modify the objects passed to it.
The type Type must be such that an object of type InputIt can be dereferenced and then implicitly converted to Type.
Returns
Iterator to the first element satisfying the condition or last if no such element is found.
